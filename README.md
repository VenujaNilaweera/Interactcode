# Interactcode

Initial project setup for Interactcode.

## Getting Started

1. Clone the repository.
2. Create and activate a virtual environment.
3. Run the starter Python script.
4. Commit and push your changes.

## Python Setup

This project now includes a local virtual environment at `.venv`.

### Windows PowerShell

```powershell
.\.venv\Scripts\Activate.ps1
```

### Run the project

```powershell
python main.py
```

If you prefer not to activate the environment, run:

```powershell
"d:/project/interactive coding/Interactcode/.venv/Scripts/python.exe" main.py
```
